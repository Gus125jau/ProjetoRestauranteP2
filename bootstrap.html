Organização do Layout com Bootstrap 4 Grid

Estrutura Básica do Grid

O sistema de grid do Bootstrap 4 é baseado em 12 colunas e utiliza classes responsivas para diferentes tamanhos de tela.

Estrutura HTML básica:

<div class="container">
    <div class="row">
        <div class="col-md-4">Conteúdo</div>
        <div class="col-md-4">Conteúdo</div>
        <div class="col-md-4">Conteúdo</div>
    </div>
</div>

Breakpoints e Responsividade

Classe Breakpoint Largura Comportamento
.col- Extra small <576px Mobile (celulares pequenos)
.col-sm- Small ≥576px Mobile (celulares maiores)
.col-md- Medium ≥768px Tablets
.col-lg- Large ≥992px Desktops
.col-xl- Extra large ≥1200px Desktops grandes

Estratégia de Layout Responsivo

Para criar um layout responsivo eficiente:

  1. Comece pelo mobile (mobile-first)
  2. Use classes sem prefixo (.col-) para layouts que devem ser iguais em todos os dispositivos
  3. Adicione classes com prefixos (.col-md-, .col-lg-) para ajustar o layout em telas maiores
  4. Utilize .row para agrupar colunas e garantir o alinhamento correto
  5. Combine classes para diferentes breakpoints (ex: col-md-6 col-lg-4)
